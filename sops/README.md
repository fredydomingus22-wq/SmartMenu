Esta pasta contém os procedimentos operacionais padrão do projeto SmartMenu.

| SOP | Descrição |
|-----|-----------|
| [resolucao_erro_500.md](./resolucao_erro_500.md) | Procedimento para resolver erros 500 (Internal Server Error) entre Frontend e Backend |

## Quando criar um novo SOP

Crie um novo SOP sempre que:
- Um erro recorrente for identificado e resolvido
- Um processo de configuração/deploy precisar ser documentado
- Uma integração complexa for implementada
